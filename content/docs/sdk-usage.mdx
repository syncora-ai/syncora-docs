# SDK Usage

Integrate our platform into your applications using our official SDKs.

## Available SDKs

### JavaScript/TypeScript

```bash
npm install @syncora/sdk
```

### Python

```bash
pip install syncora-sdk
```

### Go

```bash
go get github.com/syncora/sdk
```

## Quick Start

### JavaScript Example

```javascript
import { SyncoraClient } from "@syncora/sdk";

const client = new SyncoraClient({
  apiKey: "your-api-key",
  environment: "production",
});

// Fetch data
const data = await client.getData({
  source: "users",
  limit: 100,
});
```

### Python Example

```python
from syncora import SyncoraClient

client = SyncoraClient(
    api_key="your-api-key",
    environment="production"
)

# Fetch data
data = client.get_data(
    source="users",
    limit=100
)
```

## Advanced Usage

### Error Handling

```javascript
try {
  const result = await client.processData(data);
} catch (error) {
  console.error("API Error:", error.message);
}
```

### Configuration Options

- Environment settings
- Retry policies
