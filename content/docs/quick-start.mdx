# Quick start

Welcome to Syncora! This guide will help you get up and running in just a few minutes.

## 1. Access the Platform

Visit [Syncora Platform](https://app.syncora.ai) to access our web application.

## 2. Create Your Account

### Sign Up

1. Click the **"Sign Up"** button on the homepage
2. Enter your email address
3. Create a strong password
4. Verify your email address
5. Complete your profile setup

### Login

1. Navigate to the login page
2. Enter your email and password
3. Click **"Sign In"**

## 3. Generate Your First API Key

1. After logging in, go to **Settings** â†’ **API Keys**
2. Click **"Generate New API Key"**
3. Give your key a descriptive name
4. Copy the generated key and store it securely

## 4. Install the SDK

```bash
npm install @syncora/sdk
```

## 5. Create Your First Synthetic Dataset

```javascript
import { SyncoraClient } from "@syncora/sdk";

const client = new SyncoraClient({
  apiKey: "your-api-key-here",
});

const dataset = await client.generateData({
  schema: {
    users: {
      fields: {
        name: "string",
        email: "email",
        age: "number",
      },
      count: 1000,
    },
  },
});

console.log(dataset);
```
